# 競技プログラミング用の開発環境

## 概要

競技プログラミング用の開発環境を構築するためのリポジトリです。

以下の言語に対応しています。

- Python
- Pypy
- C++

|        | Python    | Pypy            | C++       |
| ------ | --------- | --------------- | --------- |
| 実行    | Ctrl + F5 | `make run-pypy` | `make -B` |
| デバッグ | F5        | 非対応          | F5        |
